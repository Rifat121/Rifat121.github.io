Multi-Label
Sentiment Analysis

Machine Learning Project

A machine learning model that may filter out sarcastic product reviews and then label that particular review using Multi-label Sentiment Analysis. I have trained the model in five different sentiments i.e. Happy, Sad, Neutral, Admiration, Angry. This Model is Live in this link

During the training process of this model, I had faced issues finding the proper dataset So, I had to manually create a dataset for this. After this, I had tried to use several vectorizers for the feature extraction i.e. CountVectorizer, TF-IDF Vectorizer I had also analysed my result using Senticnet5. After that, I had applied several machine learning algorithms and compared the result among them. I had applied

  • Support Vector Machine (SVM)
  • Random Forest Algorithm
  • Logistic Regression
  • Naive Bayes Algorithm
I had also considered using OnevsRestClassifier for multi-label classification.
